This tool allows for removal of unwanted sequences from .fasta files, used for creation of phylogenetic trees.
Use this script by running a command:

'''bash
python3 Gardener.py -n path/to/nexus/file -f path/to/fasta/file
'''
For additional information read Gardener_nexus_instruction.pdf run

'''bash
python3 Gardener.py -h
'''
